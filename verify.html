üíª Usage Examples

JavaScript (Fetch API):

const licenseKey = 'ABCD-1234-EFGH-5678';
const accountId = 12345;

fetch(`https://license-admin-system.vercel.app/api/verify?license_key=${licenseKey}&account_id=${accountId}`)
  .then(response => response.json())
  .then(data => {
    if (data.status === 'valid') {
      console.log('License is valid!');
      console.log('Expires in:', data.data.days_remaining, 'days');
    } else {
      console.log('License is invalid:', data.message);
    }
  });

cURL:

curl "https://license-admin-system.vercel.app/api/verify?license_key=ABCD-1234-EFGH-5678&account_id=12345"

Python:

import requests

license_key = 'ABCD-1234-EFGH-5678'
account_id = 12345

response = requests.get(
    'https://license-admin-system.vercel.app/api/verify',
    params={'license_key': license_key, 'account_id': account_id}
)

data = response.json()
if data['status'] == 'valid':
    print(f"License valid! Days remaining: {data['data']['days_remaining']}")
